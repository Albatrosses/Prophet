# Prophet(先知)
This is Hybrid Crawler system, collect daily information for our life.

# Environment
node > 6.0<br/>
npm > 6.0<br/>
yarn > 1.2.0

# Technologies
1. Express.js
2. Puppeteer
3. Graphql(Apollo)
4. Request.js
5. Cheerio
6. Mongodb
7. Typescript

# Init project
```bash
yarn install
```

# Dev project
```bash
yarn dev
```

# Run project on production
```bash
yarn server
```
