# Prophet Frontend(React)
This is Frontend for Hybrid Crawler system.

# Environment
node > 6.0<br/>
npm > 6.0<br/>
yarn > 1.2.0

# Technologies
1. React
2. Typescript
3. Graphql(Apollo)
4. Antd
5. Sass
6. styled-components
7. Jest

# Init project
```bash
yarn install
```

# Dev project
```bash
yarn dev
```

# Build project
```bash
yarn build
```
